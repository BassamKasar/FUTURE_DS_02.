# SaaS Customer Retention & Churn Analysis (2026)

## Project Overview
This project analyzes a B2B SaaS sales dataset to identify churn patterns and provide actionable retention strategies. 

### Key Business Question
"Why are customers leaving, and which segments are at risk?"

## Methodology
1. **Data Engineering (Excel):** Engineered `Cohort Months` and `Last Activity` dates.
2. **Churn Definition:** Defined "Churn" as any customer with >90 days of inactivity.
3. **Analysis (Power BI / Excel):** Performed cohort analysis and segmented churn by Industry and Customer Segment.

## Key Insights
- **Overall Churn Rate:** 2.36% (Healthy, but concentrated).
- **Highest Risk Industry:** Consumer Products (11.1% churn).
- **Most Stable Segment:** Enterprise (0% churn).

## Recommendations
- Implement a 60-day "At-Risk" email trigger for SMB customers.
- Create industry-specific feature webinars for the Energy and Consumer sectors.
